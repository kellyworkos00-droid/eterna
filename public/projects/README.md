# Portfolio Project Images

## Instructions for Adding Before/After Images

To display actual before and after images in your portfolio, add your project images to this folder with the following naming convention:

### Required Images:

1. **before1.jpg** - Living Room (Contemporary Living Space)
2. **after1.jpg** - Living Room (Contemporary Living Space)
3. **before2.jpg** - Kitchen (Gourmet Kitchen)
4. **after2.jpg** - Kitchen (Gourmet Kitchen)
5. **before3.jpg** - Bedroom (Master Suite Retreat)
6. **after3.jpg** - Bedroom (Master Suite Retreat)
7. **before4.jpg** - Office (Executive Home Office)
8. **after4.jpg** - Office (Executive Home Office)
9. **before5.jpg** - Commercial (Boutique Hotel Lobby)
10. **after5.jpg** - Commercial (Boutique Hotel Lobby)
11. **before6.jpg** - Living Room (Open Concept Living)
12. **after6.jpg** - Living Room (Open Concept Living)

### Image Specifications:

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080px (16:9 aspect ratio)
- **Max File Size**: 2MB per image (for optimal loading)
- **Quality**: High resolution, well-lit, professional photos

### Tips for Best Results:

1. Take photos from the same angle for before/after pairs
2. Use similar lighting conditions
3. Ensure the same framing and composition
4. Use high-quality, professional photographs
5. Optimize images before uploading (compress without losing quality)

### Current Status:

The slider currently shows placeholder designs with gradient backgrounds and text overlays. Once you add the actual images here, they will automatically replace the placeholders.
